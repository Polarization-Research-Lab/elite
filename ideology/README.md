# How it works:

- Data is downloaded with `download.py`
- R scripts used to run `ggum` functions to calculate ideology scores
- Processed with `process.py` (sends data to internal lab database)
- Dotplots created with R scripts (dotplot images are used in the online dashboard)
- `update` runs all scripts
